これは社員管理システムです
フロントエンドは勉強のため、uiのライブラリ使いませんでした。

リクエストはaxiosしております。
サーバーの言語は　java,フレームワークはspring bootです。
バッグ探すのため、ユーザーチェック関数を閉めました。

一部の選択肢画面項目の内容はデータベースに取るが設定しております。データベースはパソコンのシステムが崩壊ので、なくなりました。
